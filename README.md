# Genome Platform

A genome intelligence platform for health insights and genetic analysis.

## Files

- `genome-platform-static.html` — Static HTML genome analysis interface
- `genome-platform-v2.html` — Version 2 with enhanced features
- `Genetic Health/` — Analysis scripts and health data
- `AncestryDNA.txt` — Sample genome data (raw DNA file)

## Usage

Open `genome-platform-v2.html` in a browser for the full interface.
